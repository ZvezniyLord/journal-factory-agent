# Короткий звіт

Створено кандидат `Journal Skill v4.0.0-draft.1`.

Ключові зміни:

1. Перші та остання сторінки ETALON оголошені непрозорими незмінними зонами. Runtime працює тільки з точним `JOURNAL_CONTENT_SLOT`.
2. У slot дозволена одна послідовність: таблиця змісту -> один розрив сторінки -> тіло чернетки.
3. Stage 1 переносить статті по черзі та змінює лише 11 pt і міжрядковий інтервал 1.0. Абзаци, текст, таблиці, рисунки й розриви не ремонтуються.
4. Stage 2 не редагує DOCX. Він створює JSON-розмітку технічних зон: UDC/DOI, люди, ступені, звання, посади, установи, місця, ORCID, назва, анотація, ключові слова, таблиці, рисунки, джерела та unknown.
5. Проаналізовано 79 доступних статей конференцій 133/136/138. Поріг 90% пройшли лише сімейства `УДК/UDC` (91.1%) і заголовків джерел/references (93.7%). Анотації, ключові слова, висновки та DOI не оголошені обов’язковими.
6. Додано JSON Schema, приклади, acceptance checklist, marker database і changelog.
7. Активний production release не змінено.
