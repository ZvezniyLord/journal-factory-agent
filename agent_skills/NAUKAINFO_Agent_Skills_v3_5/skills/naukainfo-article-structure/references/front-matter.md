# Front-matter rules

Expected logical order: DOI (optional) → UDC → authors/affiliations → title → body → references.

The title can span multiple consecutive paragraphs. Join candidate lines for comparison but preserve original Word paragraphs unless an approved repair explicitly changes them.

Do not pull into the header:
- citations like `[1]`;
- long prose;
- bullets;
- table/figure captions;
- annotations/keywords;
- body paragraphs;
- references.

Contact data in the header is removable service content when the configured policy permits it; its removal is not critical content loss.
